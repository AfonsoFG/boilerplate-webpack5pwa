## How to use
```
 $ git clone git@github.com:AfonsoFG/boilerplate-webpack5pwa.git
```

```
 $ npm install
```

### Chose a npm run option
```
 $ npm run dev
 $ npm run build
 $ npm run serve
 $ npm run lint
 $ npm run prettify
 ```
