# PWA Webpack Boilerplate for fast project kickstart

Kickstart fast a simples ECMAScript 2015 project without any framework

## Includes
- Webpack 5
- Babel 6
- Eslint 8
- SASS
- Prettier

## PWA ready
Already has everything needed to be a PWA

## How to use
```
 $ npm clone ....
 $ npm install
 $ npm run dev
```
